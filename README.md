Bootstrap Project:
This project is a responsive e-commerce web application developed using HTML, CSS, JavaScript, and Bootstrap.  
It demonstrates the use of Bootstrap components and grid system to create a clean, user-friendly, and mobile-responsive interface.

Features:
- Responsive design using Bootstrap
- Clean and simple UI
- Bootstrap components like navbar, cards, buttons, and forms
- Cross-browser compatibility

Technologies Used:
- HTML5
- CSS3
- JavaScript
- Bootstrap

Purpose:
This project was developed to practice and demonstrate frontend web development skills using Bootstrap.
